# Contributing

Thank you for your interest! This project is part of the [SuperInstance](https://github.com/SuperInstance) ecosystem.

## Quick Start

```bash
git clone https://github.com/SuperInstance/capitaine-1.git
cd capitaine-1
npm install
npm test
```

## Guidelines

- Follow existing code style
- Add tests for new features (use the simple assert pattern in `tests/`)
- Update documentation when changing behavior
- See [org-level contributing guide](https://github.com/SuperInstance/.github/blob/main/CONTRIBUTING.md) for full details

## Reporting Issues

Use [GitHub Issues](https://github.com/SuperInstance/capitaine-1/issues) for bugs and feature requests.
