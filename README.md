## ⚓ Mission

This repository is the **HackerNews announcement point** — the zero‑shot encounter with the Lucineer fleet. It must be:

- **Educational** — concepts/, tutorials/, fleet/ folders with clear diagrams and explanations.
- **Beautiful** — the landing page is the first impression; it must be clean, fast, and inviting.
- **Honest** — real stats, real repos, real capabilities. No vaporware.
- **Marine** — vessel, captain, helm, fleet — but not overdone.

When visitors arrive, they should **understand the idea in 30 seconds** and want to explore the fleet.

---

## 🧠 The Idea

Capitaine is not a chatbot with git installed.  
It is a **repository that acts**.  

- **Body** = code in this repo  
- **Memory** = git history  
- **Nervous system** = the heartbeat cycle (regular review, update, and commit cycles)  
- **Captain** = the agent logic that decides what to do next  

Every commit is a thought. Every issue is a task. Every PR is a coordinated action with the fleet.
