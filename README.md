## ⚙️ How Capitaine Works

- **Heartbeat cycle**: Every few minutes, Capitaine reviews state (commits, issues, PRs), prioritizes tasks, and executes one action.
- **Self‑improvement**: It edits its own code, updates documentation, and coordinates with other vessels via PRs.
- **Marine metaphor**: Vessel, captain, helm, fleet — kept consistent but not overdone.
